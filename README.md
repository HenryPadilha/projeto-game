# 🎮 Jogo de Adivinhação no Terminal

## Descrição
Este é um jogo simples feito em JavaScript que roda no terminal.

O jogador precisa adivinhar um número entre 1 e 10.

## Tecnologias
- JavaScript
- Node.js
- readline-sync

## Execução

Instalar dependência:

npm install readline-sync

Executar o jogo:

node jogo.js
